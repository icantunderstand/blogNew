## 跨端的本质

一句话概括: 跨端技术是通过技术手段以较低成本和相对的稳定性实现业务快速发展

| 目标         | 考量点                                                |
| ------------ | ----------------------------------------------------- |
| 动态化诉求   | 1. 包体积问题(下载/空间占用) 2. 业务快速迭代/动态更新 |
| 人效         | 一次开发'多'端使用,提升人效                           |
| 性能         | 解决性能问题,提升业务收益                             |
| 技术基建能力 | 技术储备与探索                                        |
