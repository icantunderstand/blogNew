## Server Component

更小的JavaScript bundle: Server Component不需要发送到客户端
更好的性能: 服务器端数据获取，减少客户端网络请求
渐进式增强: 可以混合使用Server和Client Component
SEO友好: 内容在服务器端  

流式支持：支持流式渲染 的意思是不断的创建script标签
